## 우선순위 및 일정

### Phase 1 (MVP) - 4주
- **Week 1**: 프로젝트 설정, 데이터베이스 구현
- **Week 2**: 기본 UI 구현, CRUD 기능
- **Week 3**: 차트 구현, 상태 관리
- **Week 4**: 테스트, 버그 수정, 최종 정리

### Phase 2 - 3주
- **Week 1**: 알림 시스템, 데이터 내보내기
- **Week 2**: 다크모드, 성능 최적화
- **Week 3**: 테스트, 버그 수정

### Phase 3 - 4주
- **Week 1-2**: 고급 분석 기능
- **Week 3-4**: 소셜 기능, 웹 버전
